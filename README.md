# Vue.js 3 Complete Masterclass by Vladilen Minin - Practice & Projects

This repository contains my hands-on code implementations, application architectures, notes, and practical exercises completed during the comprehensive **"Vue.js 3 Course"** by Vladilen Minin.

The purpose of this repository is to track my journey from mastering core reactive principles to building high-performance, progressive web applications using the latest standards of the Vue 3 ecosystem.

## 🚀 Repository Structure

The code is organized by structural blocks matching the foundational stages of the masterclass:

```text
.
├── README.md
├── 01-introduction/                # Framework comparisons (Vue vs React vs Angular) and Vue 3 core changes
├── 02-vue-basics/                  # Core concepts: data, methods, computed properties, and reactive states
├── 03-advanced-directives/         # Loops, custom directives, modifiers, and dynamic list filtering
└── 04-vue-internals/               # Under the hood: Virtual DOM, render(), h() functions, and JavaScript Proxies
```
